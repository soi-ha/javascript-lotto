## 구입 전 / 후 `.lotto-paper`

- 구입할 금액 입력
- 구매 내역 및 로또 번호 출력
- 당첨 번호, 보너스 번호 입력
- 결과 확인 플로팅 버튼

## 당첨 통계 모달

- 당첨 통계 타이틀
- 당첨 통계 표
- 총 수익률 결과
- retry 플로팅 버튼
